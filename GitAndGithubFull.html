Key Commands for Committing

  • Ensure that you’re on the correct folder location
  • git status (Check which branch you’re on)
  • git stash (Stash your changes before pulling)
  • git pull
  • git stash apply
  • git add .
  • git checkout -b [“Branch Name”]
  • git commit -m [“Message”]
  • git push origin [Branch Name]
